Wooden Toy Festival

题目大意:

给定 n 个数 $\alpha_i$, 我们可以选择三个数$k_1,k_2,k_3$,使得max($\sum_{i=1}^n \sum_{j=1}^3 |a_i - k_j|$)最小。求出这个最大值。

问题分析:

求最小的最大值我们通常采用二分，二分关键是check()函数的写法。我们可以将问题转化为给定一个mid, 是否满足 max($\sum_{i=1}^n \sum_{j=1}^3 |a_i - k_j| $ ) <= mid。我们考虑将原数组排序，这样$k_1$ 负责 最大的前半部分（0 - i）, $k_3$ 负责最大的后半部分(j - n - 1 )，再检查$k_2$ 负责的部分是否合法即可。

Minimize Permutation Subarrays

题目大意:

给定n个数组成的排列，我们可以任意选择排列中的两个数进行交换，要求交换后的排列的子排列数量最少。

问题分析:

原排列的子排列至少有两个,即 [1] 和 它本身，我们发现其他的子排列至少包含1，2这两个元素，也就必然包含这两个数之间的所有数。于是我们可以考虑 n 的位置，若 n 在1，2之间，那么原排列的子排列就必然只有两个，于是我们只需要使得n处在1，2之间即可。

Lamps

题目大意:

给定n个灯泡，每个灯泡有两个参数a, b ；三种状态 打开，关闭，坏掉。初始所有灯泡关闭，你每次操作可以打开一个灯泡，获得分数$b_i$。记录打开的灯泡数量为x, 所有参数a >= x的灯泡会坏掉。要求所得分数最大。

问题分析:

我们按照参数a的升序进行考虑，记参数为a的灯泡数量为$cnt_a$，易知，所有参数为a-1的灯泡会比a先坏掉。所以参数为a的灯泡是否会在选择之前坏掉，只需要考虑参数为a的灯泡的数量。考虑$cnt_a <= a$，这时我们可以选择所有的参数为a灯泡，若$cnt_a > a$，则我们选择b最大的a个灯泡。

移动机器人

题目描述:

给定nums数组表述机器人开始的坐标，坐标无限长，给定字符串指令s，s[i]表示第i个机器人移动的方向，两个机器人相撞会交换方向，求执行指令d秒后，所有机器人两两之间的距离之和即$ans = \sum_{i=1}^n \sum_{j=i+1}^n |a_i-a_j|$ 。所有机器人无差别。

问题分析:

由于所有机器人等价，于是碰撞改变方向可以不考虑，于是d秒后所有机器人的位置很容易得出，所求ans暴力求解O($n^2$)的时间复杂度，这里将位置数组排序，第i个机器人a[i]他对于ans的贡献为$\sum_{j = 0}^{i-1} a_i - a_j = i * a_i - (a_0 + a_1 + ······a_{i-1}) $，于是可以在O(n)的时间复杂度解决。
